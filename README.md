# argo.clusters.zjusct.io

中文文档见 [服务部署工具 - ZJUSCT](https://www.zjusct.io/operation/k8s/svc/tools.md)。

This repository contains the configuration files for the Argo CD application deployed in the Kubernetes cluster of ZJUSCT. The Argo CD application is used for continuous deployment and GitOps practices in the cluster.
